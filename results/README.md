# Results

This folder keeps only the lightweight result tables used by the public README.

- `789_full_repo_main_results.csv`: Full 789-question MemLens results scored with eval_v2.1.
- `789_full_length_curve.csv`: Length curve on 789 questions across 32K, 64K, 128K, and 256K contexts.

Large prediction payloads, traces, prompts, model outputs, and intermediate run folders are intentionally excluded from the public repository.
